# 1. 명제와 연산자

## 명제란?
- 참이나 거싲으로 진리를 구분할 수 있는 문장
- 0 또는 1만 가지는 컴퓨터 메모리처럼 항상 참과 거짓 둘 중 하나의 값만 가짐
- 여러 개의 명제 조합 가능

## 연산자로 명제 다루기
- Not : ¬
    - 명제가 거짓일 때 참
- And(논리곱) : ∧
    - 두 명제 모두 참일때 참
- Or(논리합) : ∨
    - 두 명제 중 하나 이상 참일 때 참
- XOR(배타적 논리합) : ⊕
    - 두 명제 중 하나만 참일 때 참
- Implication(함축) : →
- Biconditional(일치) : ↔

# 2. 역, 이, 대우

## 진리표

|p|q|~p|~q|p→q|~p→~q|q→p|~q→~p|
|---|---|---|---|---|---|---|---|
|T|T|F|F|T|T|T|T|
|T|F|F|T|F|T|T|F|
|F|T|T|F|T|F|F|T|
|F|F|T|T|T|T|T|T|

## 역, 이, 대우란?

명제|역|이|대우
---|---|---|---
p → q|q → p|~p → ~q|~q → ~p

- 조건명제에서 사용
- 하나의 명제를 변형해 표현
- 증명하기 어려운 명제는 대우를 이용해 증명할 수 있음

# 3. 동치

## 동치란?
- '논리적으로 일치한다'는 의미를 지님
- 흔히 동치는 같은 의미를 가진 더 쉬운 명제를 발견하는 데 사용
- 법칙에 다양한 종류가 있음
- 복잡해 보이는 합성 명제도 간단한 명제로 바꿀 수 있음
